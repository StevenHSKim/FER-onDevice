# FER-onDevice

## 폴더 구조
```bash
.
├── inference_utils.py          # 추론에 필요한 모델, 데이터로드 등
├── run_inference.py            # 데스크탑 환경에서 추론 실행
├── jetson_run_inference.py     # Jetson Nano에서 추론 실행
├── run.sh                      # 데스크탑 환경에서 추론 실행 스크립트
├── jetson_run.sh               # Jetson Nano에서 추론 실행 스크립트
└── visualize_results.py        # 결과 시각화
```
